# Year-End-Project

Our IB CS HL year end project is an easy to use flashcard app called FlashIQ. The main problem FlashIQ addresses is that it has become significantly harder to effectively study in a convenient matter, as paper index cards are hard to carry around and manage while higher quality quiz apps are harder to access; some require users to pay money to access all the features. FlashIQ allows users to easily study various subjects no matter where they are, giving them the option to create various flashcard decks, test themselves to assess their knowledge, and so much more!
